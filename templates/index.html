Visual deepfake detection

Select a video file.

A face must be clearly visible. Files are processed locally and deleted after analysis.

01

Face crop

MediaPipe BlazeFace picks the highest-confidence face per frame, padded by 20%. If no face is detected on any sampled frame, the request is rejected before inference — non-face crops would produce a meaningless score.

02

Per-frame inference

Fine-tuned EfficientNet-B0 (top 80 layers unfrozen, BN kept in inference mode) maps each 224×224 face crop to a fake probability. 10 uniformly sampled frames per video.

03

Three-way verdict

Median aggregation gives one video-level score. A fixed band [0.40, 0.85] separates confident REAL / FAKE from UNCERTAIN. The optimised threshold (0.79) sits inside the band, so semantics stay stable across retrainings.
